# Inverted-Search-in-C

## 📌 Overview
Inverted Search (Inverted Index) system built in C using Hash Tables and Linked Lists. The project indexes words from multiple text files and enables fast word searching, database creation, updating, saving, and display operations. It demonstrates concepts like hashing, dynamic memory allocation, file handling, linked lists, and modular programming.

This project provides a basic understanding of how search engines organize and retrieve data efficiently.

## ⚙️ Working Flow

1. Read input text files  
2. Validate files  
3. Extract words from files  
4. Generate hash index  
5. Store words using linked lists  
6. Perform fast search operations  
7. Display and save database
